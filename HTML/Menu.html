Punto 1

Realizar una aplicación web que permita contar la cantidad de caracteres de un texto ingresado por el usuario.


Punto 2

Realizar una aplicación web que permita identificar si el valor ingresado por el usuario es un tipo de dato texto o número.


Punto 3

Realizar una aplicación web que permita identificar si una persona es mayor de edad. Debe pedir el nombre y la edad.





Punto 4

Realizar una aplicación web que permita saber si un usuario pasa o pierde una materia. La aplicación debe pedirle al usuario que ingrese el nombre de la persona, el nombre de la materia, y debe pedirle 3 notas. El algoritmo debe calcular el promedio de las 3 notas y determinar con ese promedio si pasa o no la materia. Si el promedio es menor a 3.0, debe indicar que perdió la materia, pero si es igual o superior a 3.0 debe indicar que pasó la materia. Debe usar colores como el rojo o el verde para indicar que si se pasa o se pierde la materia.











Punto 5

Realizar una aplicación para identificar números pares e impares. El número debe ser ingresado por el usuario y debe ser un número entero positivo.



Punto 6

Realizar una aplicación que permite buscar contenido en un texto. El usuario debe ingresar un texto (oración, párrafos, etc.) y debe ingresar el texto a buscar, El sistema debe mostrar o resaltar las coincidencias encontradas en el texto.





Punto 7

Realizar una aplicación que permita saber si un número es divisible por otro. El usuario debe ingresar dos números enteros positivos a y b, y decir si son divisibles entre sí.





Punto 8

Realizar una aplicación que permita identificar una lista de números enteros, ingresados por el usuario y separados por coma, cuales son números pares e impares.
